# MapsApp

Pasos para ejecutar el proyecto:

1. Clonar el repositorio
2. Instalar las dependencias con el comando: `npm install`
3. Generar el archivo .env basado en el archivo .env.template
4. Ejecutar el comando: `npm run set-envs` para generar los archivos ('./src/environments/environment.ts' y './src/environments/environment.development.ts')
